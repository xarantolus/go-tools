Date Formatter

Date formatting in Go is different from most programming languages. Instead of using a format like yyyy-MM-dd HH:mm:ss or %Y-%m-%d %H:%M:%S, you have to write the date Mon Jan 2 15:04:05 -0700 MST 2006 Source.
You can put the date format on the right in your code like this: var formattedDateString = timestamp.Format("2006-01-02 15:04:05 (Monday)"))
